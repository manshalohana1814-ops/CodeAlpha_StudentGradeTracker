# Student Grade Tracker 🎓

This is a Java-based Student Grade Tracker project developed during my CodeAlpha Internship.

## 🚀 Features
- Add student details
- Enter subject marks
- Calculate grades automatically
- Simple GUI interface
- Input validation system

## 🛠️ Tech Stack
- Java
- Swing (GUI)

## 📌 How to Run
1. Clone the repository
2. Open in any Java IDE (NetBeans / IntelliJ / Eclipse)
3. Run the main file
